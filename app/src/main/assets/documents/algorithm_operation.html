Умное распределение устройств по группам и фазам

Алгоритм VoltHome автоматически формирует группы, подбирает защиту и равномерно распределяет нагрузку по фазам A/B/C с учётом коэффициента спроса и cos φ.

Анализ проекта

Алгоритм начинает с анализа всех помещений и устройств:

  • Данные по каждому устройству (мощность, тип, напряжение)
  • Определение «особых зон» (ванная, кухня, улица)
  • Учёт коэффициента спроса и cos φ
  • Выявление приборов, требующих выделенной линии
cos φ demandRatio Влажные зоны

HEAVY_DUTY‑линии

Для энергоёмких приборов создаются выделенные группы:

  • Кондиционеры, водонагреватели, плиты, варочные панели
  • Техника мощнее ~2000 Вт или с пусковыми токами
  • Подбор усиленной защиты и кабеля

Типичные параметры: автомат 25–32 А (D/C), кабель 4–6 мм², селективность по УЗО.

Группы освещения

  • Отдельные линии для света (10 А, кабель 1.5 мм²)
  • Разделение основного и декоративного освещения

Розеточные группы

  • Автоматы «C» 16 А, кабель 2.5 мм²
  • Контроль количества и суммарной нагрузки
  • Автоматическое разбиение при перегрузке

Балансировка по фазам

После формирования групп выполняется равномерное распределение по фазам A/B/C:

  • Считаем ток группы: I = P · demandRatio / (U · cos φ)
  • Жадное распределение с выравниванием суммарных токов фаз
  • Контроль допустимого дисбаланса (≈ до 15%)
  • Перераскладка крупных HEAVY_DUTY‑линий для баланса

Защита и безопасность

  • УЗО 30 мА для ванных, кухонь, уличных розеток
  • Проверка автомата/кабеля по току и пусковым нагрузкам
  • Селективность вводного аппарата

Формирование результатов

  • Перечень групп с устройствами
  • Технические параметры (ток, автомат, кабель)
  • Фаза каждой группы (A/B/C) и баланс по фазам
  • Отчёты по помещениям, пометки об УЗО
