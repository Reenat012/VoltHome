2. Обработка данных

Приложение VoltHome не собирает и не передает персональные данные пользователей в смысле Федерального закона РФ №152-ФЗ «О персональных данных».

Все вводимые пользователем данные (например, имена комнат, названия проектов, заметки) обрабатываются только локально и не передаются разработчику или третьим лицам.

- Локальное хранение

Данные сохраняются исключительно в памяти устройства в базе данных, защищённой средствами ОС Android.

- Отсутствие передачи данных

Приложение не использует интернет-соединение, не обращается к удалённым серверам и не содержит аналитических или рекламных SDK.

- Системные разрешения

Приложение не запрашивает доступ к персональным данным. При экспорте/резервном копировании проектов используется локальное хранилище устройства (Scoped Storage / MediaStore).

3. Гарантии безопасности

- Шифрование данных

База данных защищена средствами ОС Android:

  • Шифрование на основе ключей устройства (AES-256 через Android Keystore);
  • Хранение в изолированном sandbox-каталоге /data/data/ru.mugalimov.volthome/databases.

- Отсутствие доступа разработчика

Разработчик не имеет технической возможности получить доступ к данным на устройстве пользователя.

4. Данные, не относящиеся к приложению

- Email-коммуникация

Приложение содержит ссылку на email разработчика reenat056@yandex.ru. Любые данные, отправленные пользователем через почтовый клиент:

  • обрабатываются вне приложения;
  • регулируются политикой конфиденциальности почтового сервиса (Яндекс.Почта, Gmail и др.).

- Отсутствие интеграций

Приложение не интегрировано с мессенджерами, внешними API или облачными сервисами.

5. Права пользователя

Пользователь сохраняет полный контроль над своими данными:

- Удаление данных

  • Через настройки приложения («Очистить базу данных»);
  • При удалении приложения (данные автоматически стираются).

- Экспорт данных

Экспорт ограничен локальными копиями на устройстве. Передача в сторонние сервисы отсутствует.

6. Обязательства разработчика

  • Разработчик не получает, не хранит и не обрабатывает данные, созданные пользователем.
  • Архитектура приложения соответствует:
  • ст. 18.1 ФЗ-152 «О персональных данных» (локальная обработка);
  • GDPR (ст. 25 «Privacy by Design»);
  • Политикам Google Play и RuStore для offline-приложений.

7. Изменения в политике

Любые изменения будут:

  1. опубликованы на этой странице;
  2. отражены в обновлении приложения в магазинах;
  3. продублированы внутри приложения (в разделе «О приложении»).

Push-уведомления не используются, так как приложение работает оффлайн.
